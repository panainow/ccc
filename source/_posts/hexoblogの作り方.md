#### hexoでブログ作成
全くの初心者なので細かいとこは
現状、理解してないです。
実際に試して進めていきます。
	
ターミナルを使います。

<img src="terminal.jpg" alt="" title="hexoblog" width="400">

コレですね。

それでは、実際に作成していきます。
ローカルでブログ用のディレクトリを作成し、インストールします

``` npm install -g hexo-cli
```

空のディレクトリを作成し、そこで

``` hexo init
```

します。

そこで、

```npm install
```

これですでにローカルに出来上がり。

``` hexo s 
```

で確認。
